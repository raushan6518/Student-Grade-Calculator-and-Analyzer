# Student Grade Calculator and Analyzer

## Project Overview
A comprehensive Python application for calculating and analyzing student grades, SGPA, and CGPA according to VIT grading system. This tool helps students track their academic performance and plan future semesters.

## Features
- **Add Semester Grades**: Input courses, credits, and marks
- **SGPA Calculation**: Calculate Semester Grade Point Average
- **CGPA Calculation**: Compute Cumulative Grade Point Average across all semesters
- **Future Grade Predictor**: Determine grades needed to achieve target CGPA
- **Academic Reports**: Generate detailed student academic reports
- **Data Persistence**: Save and load student data automatically

## VIT Grading System
| Grade | Marks Range | Grade Point |
|-------|-------------|-------------|
| S     | 90-100      | 10          |
| A     | 80-89       | 9           |
| B     | 70-79       | 8           |
| C     | 60-69       | 7           |
| D     | 50-59       | 6           |
| E     | 40-49       | 5           |
| F     | Below 40    | 0           |
